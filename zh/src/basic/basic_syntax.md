# 基本语法

# 注释符

`#`与`//`是注释符，该行后面内容视作注释内容被忽略

```python
>> # this is a comment!
>> // this is a comment!
```

# 行尾结束符

`;`是行尾结束符，单行的时候可以忽略


# let语句

`let`语句可以定义标志符、函数等，所有变量的定义都通过`let`语句, 变量名必须是字母或`_`开头，由字母、数字和`_`组成。

# 赋值语句

通过`let`定义的变量可以通过赋值语句直接赋值:

```javascript
let i = 0;

i = 5; // OK

j = 4; // error: undefiend variable j
```

# fn语句

`fn`关键字用于定义函数。

# return语句

`return`关键字用于返回值，有时候它可以省略，隐式返回。

`return`语句可以在任何地方执行，不局限于函数内部。
