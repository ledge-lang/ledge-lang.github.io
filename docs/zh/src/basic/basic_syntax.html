基本语法

基本语法

注释符

#与//是注释符，该行后面内容视作注释内容被忽略

>> # this is a comment!
>> // this is a comment!

行尾结束符

;是行尾结束符，单行的时候可以忽略

let语句

let语句可以定义标志符、函数等，所有变量的定义都通过let语句, 变量名必须是字母或_开头，由字母、数字和_组成。

赋值语句

通过let定义的变量可以通过赋值语句直接赋值:

let i = 0;

i = 5; // OK

j = 4; // error: undefiend variable j

fn语句

fn关键字用于定义函数。

return语句

return关键字用于返回值，有时候它可以省略，隐式返回。

return语句可以在任何地方执行，不局限于函数内部。
